# Linux Commands

### show current directory
```python
pwd
```


### Change directory
```python
cd .. ---> 1 level up
cd . ----> current directory
cd ../.. -> 2 levels up
cd / ----> go to root
cd ~ ----> go to home directory
cd - =====> go to previous directory
```

### Path types
```python
cd /home/users/projects --> Absolute path

cd ../projects ---> Relative path
```


### list files
```
ls
ls -l = detailed list
ls -a = show hidden files
ls -lh = human readable sizes


dirs = show all directories
```

### create/ delete
```
mkdir test
mkdir test && cd test = create and use it
rm file.txt
rm -r foldername
```

### Tricks
```
clear 
histroy = show command history
!! = repeat last command
ctrl+r = search history


tree
```
